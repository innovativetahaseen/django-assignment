# 🎓 Django Exam Form Portal

## 📌 Project Description
This is a Django-based web application where students can:
- Login
- View dashboard
- Fill exam form
- Submit details

After submission, the data is stored in the database and a success message is displayed.

---

## 🚀 Features
- Login Page
- Dashboard
- Exam Form
- Data stored in SQLite database
- Success confirmation page

---

## 🧾 Form Fields
- Full Name  
- Course  
- Year  
- Address  
- Phone Number  

---

## 🛠️ Tech Stack
- Python
- Django
- HTML (Templates)
- Bootstrap (for UI)

---

## ⚙️ How to Run Project

Clone repository:
git clone https://github.com/innovativetahaseen/django-assignment.git

Go inside project:
cd exam_portal

Activate virtual environment:
source ../venv/bin/activate

Run server:
python manage.py runserver

---

## 🌐 Output Flow
Login Page → Dashboard → Fill Form → Success Page
